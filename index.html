User Guide

To get started with WhatsApp ChatShield, follow these steps:

  1. Download and install the extension from the Chrome Web Store.
  2. Open WhatsApp Web on your browser.
  3. Click on the ChatShield icon in your browser's toolbar.
  4. Select the features you want to use and configure them according to your preferences.
  5. Enjoy a more secure and enhanced WhatsApp experience!
